# B站视频自动总结插件

## 项目目标
本项目旨在开发一款Chrome浏览器扩展，能够自动总结B站（bilibili.com）视频内容，生成条理清晰、易于理解的总结，并在每个总结条目前加上对应的时间段，用户点击时间段可直接跳转到视频对应位置。

## 主要功能
1. 自动获取当前B站视频的标题和视频ID。
2. 自动提取视频音频内容，调用AI接口进行内容总结。
3. 生成结构化的总结，每条总结前带有时间段标签。
4. 用户点击时间段标签可直接跳转到视频对应片段。
5. 总结内容以弹窗或侧边栏形式展示，界面简洁美观，支持响应式设计。
6. 支持中英文B站视频。

## 使用方法
1. 安装本插件后，打开任意B站视频页面。
2. 点击浏览器工具栏上的插件图标，或在视频页面侧边栏点击"总结"按钮。
3. 稍等片刻，插件会自动生成该视频的结构化总结。
4. 点击总结条目前的时间段，可直接跳转到视频对应片段。

## 参数说明
- 无需用户手动输入参数，插件自动处理。
- 后续可扩展：允许用户自定义总结粒度（如每5分钟/10分钟/章节等）。

## 返回值说明
- 展示结构化总结内容，每条带有时间段和跳转功能。
- 若遇到错误（如AI接口不可用、视频无音频等），会弹窗提示。

## 开发规划
- 使用Manifest V3，采用Service Worker架构。
- 内容脚本仅在bilibili.com视频页生效，最小权限原则。
- 前端UI采用响应式设计，适配不同分辨率。
- 代码包含详细中文注释，便于理解和维护。
- 所有数据传输均使用HTTPS。
- 后续考虑支持Side Panel、Offscreen Documents等高级特性。

## 优化建议
- 后续可支持更多视频平台（如YouTube、腾讯视频等）。
- 增加多语言支持。
- 优化AI总结算法，提升准确率和可读性。
